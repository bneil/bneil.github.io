## O is for Opencode

Recently picked up opencode go, which is a low cost subscription to try out some models like kimi k2. Its been a solid
plan, as subscriptions go. At least it seems more sane than 200+ a month. In either case, its been fun trying the easy
button. Also its been fun trying to create skills from CLI skills. Really, the most interesting thing that ive enjoyed
while using the tool is this plugin called oh my openagent, that provides a bunch of persona around the requests.

Let me see, some other interesting plugins ive run across:
- plannotator: interesting in web planner, so you make a prd and then edit it in a wysiwyg interface.
- kimaki: a discord bot, just thought it was neat, im currently trying out zeroclaw as an assistant but its just 'ok'
- NickvanDyke/opencode.nvim: neovim plugin for using opencode in the buffer. I use tmux so its not sure helpful, i like
that it brings the cursor / zed / intellij feel to neovim. but honestly, i find myself doing inline code updates less. If
im using the easy button then im commiting.
